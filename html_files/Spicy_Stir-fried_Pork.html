제육볶음

🧂 재료
불고기용 앞다리살 : 500g
양배추 : 100g
양파 : 150g
당근 : 30g
대파 : 100g
청양고추 : 15g
식용유 : 3숟가락
진간장 : 4숟가락
설탕 : 1숟가락
다진마늘 : 1숟가락
고추장 : 1숟가락
굵은 고춧가루 : 3숟가락
물엿 : 1숟가락
굴소스 : 1숟가락
맛술 : 1숟가락
후추 : 0.1숟가락
👩‍🍳 레시피
1. 미리 재료 손질하기(야채 썰어두기)
2. 팬에 식용유를 두른다
3. 불고기용 앞다리살을 넣는다 (앞다리살이 없다면 얇게 썬 고기 아무거나 괜찮다)
4. 진간장4, 설탕1을 넣고 바짝 볶아준다
5. 대파, 양파, 당근, 양배추를 넣는다
6. 청양고추를 넣는다
7. 굵은 고춧가루를 넣는다
8. 고추장1, 다진마늘1, 물엿1, 맛술1, 굴소스1을 넣는다 (+후추)
9. 불을 세게 키고 야채를 빠르게 볶는다 (양파가 투명해지기 직전까지만)
10. 마지막에 참깨 뿌려주기
